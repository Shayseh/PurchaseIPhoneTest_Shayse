Purchase iPhone Automation Test

Selenium Java automation project using Maven, TestNG, Page Object Model and Excel test data

View Code on GitHub

Project Overview

This project demonstrates an automated UI test flow for purchasing an iPhone. The framework is structured using Java, Selenium WebDriver, Maven and TestNG. It follows a clean folder structure with reusable utilities, browser setup, test data handling and screenshot capture.

Tools & Technologies

Java Selenium WebDriver TestNG Maven Page Object Model Excel Test Data Screenshot Capture GitHub

Framework Structure

BaseTest

Handles test setup and driver initialization before tests are executed.

Pages

Stores page classes and locators using the Page Object Model approach.

PurchaseTest

Contains the main automated test flow for the iPhone purchase scenario.

BrowserFactory

Manages browser driver setup and helps keep browser logic reusable.

ReadXLSData

Reads Excel test data and supports data-driven testing with TestNG.

Screenshots

Captures test evidence during execution for debugging and reporting.

What This Project Shows

  • Ability to create a Selenium automation framework
  • Understanding of Page Object Model structure
  • Use of Maven for project and dependency management
  • Use of TestNG for test execution
  • Data-driven testing using Excel files
  • Screenshot handling for test evidence
